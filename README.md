# Player

## Build

### Prerequisites

- Xcode 13.0
- rust 1.55.0
- cargo lipo 3.1.1
- rust iOS targets: `aarch64-apple-ios`, `x86_64-apple-ios`

```zsh
$ git clone --recursive https://github.com/elmattic/Player.git
```

Then open Xcode project file and select the build scheme
you want (Running/Testing/Profiling).
